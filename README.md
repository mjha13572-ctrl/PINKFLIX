# 🎬 PINKFLIX

PINKFLIX is a Python desktop application built with **Tkinter** that allows users to search for movies and instantly view detailed information, posters, trailers, and user reviews.

## ✨ Features

* 🔍 Search movies by title
* 🎬 Display movie title, release year, genre, and IMDb rating
* 🖼️ Display movie posters
* 📝 Read the movie plot
* ⭐ View IMDb rating, IMDb votes, and Metascore
* 💬 Read user reviews fetched from TMDb
* 🎥 Watch the official trailer on YouTube
* 💖 Clean and user-friendly Tkinter interface

## 🛠️ Technologies Used

* Python
* Tkinter
* OMDb API
* TMDb API
* Pillow (PIL)
* Requests

## 📦 Required Libraries

Install the required libraries using:

```bash
pip install requests pillow
```

## 🚀 How to Run

1. Clone this repository.
2. Install the required libraries.
3. Run:

```bash
python movieanalyzer.py
```
